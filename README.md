# root-feedstock

[![Build & Upload](https://github.com/hep-forge/root-feedstock/actions/workflows/autoupload.yml/badge.svg)](https://github.com/hep-forge/root-feedstock/actions/workflows/autoupload.yml)
[![Anaconda Version](https://anaconda.org/hep-forge/root/badges/version.svg)](https://anaconda.org/hep-forge/root)
[![Anaconda Platforms](https://anaconda.org/hep-forge/root/badges/platforms.svg)](https://anaconda.org/hep-forge/root)

Feedstock for [root](https://root.cern/) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

ROOT: analyzing petabytes of data, scientifically.

## Install

```bash
conda install -c hep-forge -c conda-forge root
```
